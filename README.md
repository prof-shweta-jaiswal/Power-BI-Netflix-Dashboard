
# This project is a Power BI Netflix Dashboard created to analyze and visualize Netflix movies and TV shows data.

The dashboard provides insights based on genre, release year, ratings, country, and content type using interactive charts and filters. It showcases practical skills in data analysis, data cleaning, DAX, and dashboard design.

# Key Features:
- Interactive Power BI dashboard
- Netflix movies & TV shows analysis
- Genre, year, rating & country-wise insights
- Dynamic filters and slicers
- Beginner-friendly analytics project

# Tools Used:
- Power BI
- Data Analysis
- Data Visualization
